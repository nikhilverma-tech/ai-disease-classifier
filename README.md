7# AI-Based Disease Classification & Medicine Recommendation System

## Overview
Misdiagnosis and lack of access to the right medicines often delay proper treatment. 
This system uses Artificial Intelligence to classify diseases based on user symptoms, 
recommend suitable medicines (with alternatives), and help users find nearby locations 
where these medicines are available.

## Features
- Disease prediction using symptoms
- Medicine recommendation system
- Alternative medicine suggestions
- Nearby pharmacy locator
- Simple and user-friendly interface

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- AI Model: Machine Learning (Classification)
- Database: CSV / SQLite

## Future Scope
- Integration with real-time pharmacy APIs
- Doctor consultation feature
- Mobile app version

## Note
This system is for educational purposes and not a replacement for professional medical advice.
